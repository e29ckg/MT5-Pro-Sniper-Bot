# 🎯 MT5 Gemini AI Sniper Bot & Dashboard

บอทเทรดทองคำ (XAUUSD) อัตโนมัติสำหรับ MetaTrader 5 (MT5) ขับเคลื่อนด้วยสมองกล **Google Gemini 2.5 Flash** เน้นกลยุทธ์การเทรดแบบ **"ไม้เดียวจบ (One-Shot Sniper)"** ไม่มีการใช้ระบบไม้แก้ (No DCA) ปลอดภัยต่อพอร์ตลงทุน พร้อมระบบปกป้องเงินทุน (Capital Protection) และหน้าต่างควบคุม (Web Dashboard) แบบ Real-time

---

## ✨ ฟีเจอร์เด่น (Key Features)

* 🧠 **Multi-Timeframe AI Analysis:** ส่งกราฟ 2 ไทม์เฟรม (H1 เพื่อดูเทรนด์หลัก + M5 เพื่อหาจุดเข้า) ให้ AI วิเคราะห์พร้อมกัน เพื่อความแม่นยำสูงสุด
* 🧱 **Smart Market Filters:**
* ตรวจสอบ **Volume** การซื้อขายจริง เพื่อกรองจังหวะกราฟหลอก (Fakeout)
* คำนวณ **แนวรับ-แนวต้าน (Support/Resistance)** จากระดับ D1 และ H4 อัตโนมัติ
* ประเมิน **Risk:Reward Ratio (R:R)** ก่อนเข้าเทรด (หากต่ำกว่าเกณฑ์ AI จะปฏิเสธการเทรด)


* 🛡️ **Advanced Capital Protection:** ระบบเลื่อน Stop Loss ขวางหน้าทุนอัตโนมัติเมื่อราคาวิ่งถูกทาง เลือกได้ 2 โหมด:
* **ล็อกครั้งเดียว (One-Time):** ล็อกหน้าทุนแล้วปล่อยกราฟวิ่งหาเป้าหมาย (TP)
* **ขั้นบันได (Step Trailing):** เลื่อนล็อกกำไรตามระยะที่ราคาวิ่งไป (รีดกำไรสูงสุด)


* 📊 **Modern Web Dashboard:** หน้าต่างควบคุมผ่าน Streamlit (ป้องกันด้วยรหัสผ่าน) ดูกราฟราคาและ Volume แบบเรียลไทม์ พร้อมเช็คสถานะพอร์ตและความคิดของ AI ได้ทันที
* 📱 **Telegram Notifications:** แจ้งเตือนเข้ามือถือทุกสถานะ (เข้าออเดอร์, เลื่อน SL บังทุน, ปิดทำกำไร, สเปรดถ่าง)

---

## 📂 โครงสร้างไฟล์ในโปรเจกต์ (Project Structure)

* `app.py` - ระบบ Frontend Web Dashboard (สร้างด้วย Streamlit)
* `bot_backend.py` - ระบบ Backend ควบคุมการเชื่อมต่อ MT5 และ Gemini AI
* `core_db.py` - ระบบฐานข้อมูล SQLite (เปิดโหมด WAL) สำหรับเชื่อมต่อระหว่าง Frontend และ Backend
* `bot_data.sqlite` - ไฟล์ฐานข้อมูล (จะถูกสร้างขึ้นอัตโนมัติเมื่อรันระบบ)
* `trade_history.json` - ไฟล์เก็บประวัติการเทรด (ถ้ามี)

## ⚙️ สิ่งที่ต้องเตรียม (Prerequisites)

1. **Python 3.9** หรือใหม่กว่า
2. โปรแกรม **MetaTrader 5 (MT5)** (ล็อกอินบัญชีและเปิดปุ่ม **Algo Trading** ให้เป็นสีเขียว)
3. **Google Gemini API Key** (รับฟรีได้ที่ [Google AI Studio](https://aistudio.google.com/))

**ติดตั้งไลบรารีที่จำเป็น:**
เปิด Terminal/Command Prompt แล้วรันคำสั่ง:

```bash
pip install MetaTrader5 streamlit pandas plotly mplfinance google-genai Pillow requests

```

## 🚀 วิธีการใช้งาน (How to Run)

การทำงานของระบบนี้ จำเป็นต้องรัน 2 เซอร์วิสคู่กัน (ต้องเปิด Terminal 2 หน้าต่าง)

### ขั้นตอนที่ 1: เปิดระบบ Web Dashboard

1. เปิด Terminal แรกลงในโฟลเดอร์โปรเจกต์
2. รันคำสั่ง:
```bash
streamlit run app.py

```


3. เข้าไปที่เบราว์เซอร์ `http://localhost:8501`
4. ล็อกอินด้วยรหัสผ่านเริ่มต้น: `admin123` *(สามารถเปลี่ยนได้ในเมนูตั้งค่า)*
5. นำ **Gemini API Key** ไปใส่ในแถบตั้งค่าด้านซ้ายมือ และปรับแต่งกลยุทธ์ตามต้องการ

### ขั้นตอนที่ 2: เปิดระบบ Bot Backend

1. เปิด Terminal ที่สอง (ห้ามปิดหน้าต่างแรก)
2. รันคำสั่ง:
```bash
python bot_backend.py

```


3. กลับไปที่หน้า Web Dashboard แล้วกดเปิดสวิตช์ **"🚀 เปิดระบบบอท"**
4. บอทจะเริ่มทำงาน สแกนตลาด อ่านกราฟ และส่งสัญญาณเทรดตามเงื่อนไขทันที!
